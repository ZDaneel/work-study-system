# 高校学生勤工俭学系统

## 一、基本需求

高校勤工俭学基金办公室是帮助同学参加高校勤工俭学劳动的机构。人工管理中涉及对大量表格的查找，效率很低，工作人员负担很重，且易出差错。因此，该办公室需求建立一个高效、准确，可以自动生成各种表格和报表的微机管理系统。

## 二、主要功能描述

1) 学生申请表和用工计划申请表的录入和存储；

2) 按照用工计划申请表的要求自动列出候选名单；
3) 自动生成应聘合同书；

4) 自动生成酬金发放单；
5) 生成月考核表。
